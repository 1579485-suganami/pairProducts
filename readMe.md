# ポケモンおみくじ (Version 0.1, 20221104)

by Yannys, PairProject BTC-3

## はじめに

**お布施ができます！！！**
[Google先生](https://www.google.co.jp/)

## 機能
* 「お布施をする」のボタンをクリックすると、ありがたいお言葉と共に今日のラッキーポケモンを表示します。
* 905種類のポケモンに対応しています。

### 備考
* ポケモンの背景色が無駄にランダムに変わるため、ポケモンが見にくくなる可能性があります。
